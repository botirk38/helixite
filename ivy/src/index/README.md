# Index module

All index implementations live here: label indexes, property indexes, vector HNSW indexes, and key encoding utilities.

## Submodules

```
codec.rs         — binary key encoding/decoding
edges.rs         — edge adjacency index
labels.rs        — node/edge label indexes
nodes.rs         — node index orchestration
properties.rs    — property index system (with unique constraint)
vector/          — HNSW vector index
```
